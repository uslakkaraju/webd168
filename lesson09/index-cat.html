What is SVG?

SVG stands for Scalable Vector Graphics. It is an XML-based image format used to create shapes, icons, illustrations, logos, and other graphics directly in a web page. Because SVG uses code instead of pixels, the image stays sharp at any screen size.

Why use inline SVG?

Inline SVG works especially well with HTML, CSS, and JavaScript. You can target individual SVG parts with classes, change colors, animate features like eyes or whiskers, and add accessibility information such as titles and descriptions for screen readers.
